# 03_resume · 简历模板（可安全提交）

本目录为**占位模板**，不含任何个人隐私信息，可安全提交到仓库。

## 文件说明

| 文件 | 风格说明 |
|------|----------|
| `my_resume.typ` | SixtySecondsCV 双栏 |
| `resume_aero_minimal.typ` | Aero 极简单栏 |
| `resume_golden_dual.typ` | 黄金比例双栏 |
| `cv_bento.typ` | Bento 卡片流 |
| `cv_tufte.typ` | Tufte 学术旁注 |
| `cv_cli.typ` | CLI/终端极客风 |

将占位（如「您的姓名」「您的邮箱」「your@example.com」等）替换为您的真实信息后，可复制到 **03_my_resume** 目录使用。真实简历请仅保存在 `03_my_resume`，该目录已被 `.gitignore` 忽略，不会上传。
